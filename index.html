Ошибки: 0

Инструкция по использованию тренажёра фраз для письма B1

1. Выбор фразы
В выпадающем списке выберите фразу на русском языке — она будет соответствовать немецкому оригиналу, который нужно будет набрать.

2. Начало тренировки
Нажмите кнопку Начать. Немецкая фраза появится на 5 секунд, затем исчезнет, и начнётся ввод.

3. Ввод фразы
Начните вводить фразу на немецком языке:
- При каждой ошибке звучит сигнал и отображается подсказка — правильная следующая буква.
- Ведётся счётчик ошибок.
- При правильном вводе всей фразы поле окрашивается в светло-зелёный цвет и проигрывается другой звук.

4. Повторный выбор фразы
После завершения упражнения обновите страницу, чтобы снова выбрать фразу (или можете самостоятельно включить возможность выбора в коде).
